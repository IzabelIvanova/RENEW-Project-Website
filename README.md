# RENEW Project Website

This repository contains a **simple static website** for the RENEW research project. The site is built with an HTML5 UP template and extended using custom SCSS and a small amount of vanilla JavaScript. It is bilingual (Bulgarian and English) and designed to be edited directly — no build tools, frameworks, or servers are required.

## Contents

- HTML files at the root for Bulgarian pages and under `en-renew-project/` for English translations.
- SCSS source in `assets/sass/` and compiled CSS in `assets/css/`.
- JavaScript logic in `assets/js/main.js` for navigation, language toggling, and simple UI behaviors.
- Images and logos in `images/`.

## Usage

Just open any `.html` file in a browser or serve the folder via a static server (e.g., `python -m http.server`). Edit the HTML, SCSS, or JS as needed and reload.
